# 🏭 MES Production System v8.0 🚀

**MES Production** là hệ thống quản lý sản xuất thông minh, giúp theo dõi và phân bổ mục tiêu sản lượng theo thời gian thực (Real-time). Phiên bản v8.0 mang lại sự ổn định và giao diện trực quan vượt trội.

## ✨ Tính năng nổi bật trong bản v8.0
- 📊 **Dashboard Thống Kê Mới**: Hiển thị tổng quan số lượng Package, Mục tiêu và Thực tế.
- 🖼️ **Xem Trước Ảnh (Hover Preview)**: Hiển thị ảnh sản phẩm sắc nét ngay khi rê chuột.
- 🕒 **Chia Sản Lượng Thông Minh**: Tự động tính toán mục tiêu cho các khung giờ hành chính và tăng ca (OT).
- 🔍 **Bộ Lọc Nâng Cao**: Tìm kiếm và lọc mã hàng theo Line, Trạng thái hoặc OT.
- 🔐 **Bảo Mật**: Tích hợp xác thực Cloudflare và hệ thống Login hiện đại.

## 🛠️ Công nghệ sử dụng
- **Frontend**: HTML5, CSS3 (Inter Font, Font Awesome 6), JavaScript (ES6+).
- **Security**: Cloudflare Turnstile API.
- **Framework**: Bootstrap-like custom components & Google Fonts.

## 🚀 Hướng dẫn triển khai
1. Tải tệp `inde2by.html` về máy chủ của bạn.
2. Cấu hình các API endpoint (nếu có) trong phần script.
3. Chạy trực tiếp trên trình duyệt hoặc deploy qua các dịch vụ như Cloudflare Pages, GitHub Pages.

## 📱 Yêu cầu hệ thống
- Trình duyệt hiện đại (Chrome, Edge, Safari).
- Kết nối Internet để tải các thư viện CDN.
